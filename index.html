Como utilizar o ApVest

Arquivo:

O ApVest recebe um arquivo ".txt" que contém os dados para a comparação.

Formatação:

Os nomes devem estar separados por linhas, um nome em cada linha dos arquivos ".txt"

Os arquivos devem estar no seguinte formato:

Nome1
Nome2
Nome3
Nome4
Nome5

Letras maiúsculas e acentuação:

O ApVest consegue reconhecer nomes mesmo que haja as seguintes distinções nos ".txt":


Letras maísuculas:
João ↔ joão

Acentuação:
joão ↔ Joao
